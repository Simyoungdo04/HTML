요소의 영역 구분

margin : 요소의 바깥쪽 여백 영역
border : 테두리 영역
padding : 내용물 영역과 테두리 영역 사이의 거리(안쪽 여백)
content : 실제 내용물이 들어가는 영역
실제 요소가 차지하는 크기(영역) == content + border
