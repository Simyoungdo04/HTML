테두리 관련 스타일(border)

        선택자 {
            테두리 스타일
            border-위치-style : noen / dotted / dashed / solid** / double
                                groove / ridege / inset / outset

            테두리 두께
            border-위치-width : 테두리 두께 px;

            테두리 색상
            border-위치-color : 색상값;

            테두리의 모서리를 동그랗게
            border-위치-radius : 숫자 px. %;
            => 모든 위치는 생략 가능

            * 그림자
            box-shadow : 가로(X) 세로(Y) 흐림정도 번짐정도 생상;
        }
