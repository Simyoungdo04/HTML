jQuery

기존 자바스크립트만으로는(옛날에) 브라우저마다 각기 다른 코드를 작성해서
브라우저를 조작해야하는 어려움이 있었다
창시자 존 레식은 이런 문제를 해결하기 위해 JavaScript라는 언어를 이용해서
함수모음집을 만들어서 배포했다.
이것이 바로 jQuery라는 라이브러리 인 것이다.
JavaScript로 함수 많이 모아놓은 파일 == 라이브러리
-사람들은 대체 왜 그렇게 jquery에 열광 했나요
1) DOM요소와 관련된 스크립트 코드를 아주 쉽게 구현할 수 있음
2) AJAX(비동기식 통신), 이벤트 처리를 아주 쉽게 구현할 수 있음
3) jQuery와 관련된 확장 플러그인, Open API등이 많이 지원된다.

* jQuery사용하기

우리는 jQuery를 사용해서 DOM을 다룰예정 jQuery코드가 문서상 모든 요소들이 전부 다 로드된 후 에 실행할 수 있게끔 만들어 주어야함 => window.onload 

        onload : 페이지가 전부 다 로딩되고 나면 자동으로 딱 한 번 수행
        해당 HTML문서의 요소들도 전부 다 로딩되고 연동되어 있는 외부 문서들도
        전부 다 로딩된 후에 onload속성에 대입한 이벤트 핸들러가 수행되는 특징을 가짐
    
        - jQuery에서는 해당 내용을 세 가지 구문을 제공하여 해결

        1)
        jQuery(document).ready(function(){
            실행할내용;
        });

        2)
        $: jQuery키워드를 나타냄
        $(document).ready(function(){
            실행할내용;
        });

        3)
        $(function(){
            실행할내용;
        });

        jQuery에서 해당 function은 HTML문서 안에 있는 DOM요소들이 모두 로딩 되면 바로 실행
        여러번 기술 가능, 횟수 제한 없음
