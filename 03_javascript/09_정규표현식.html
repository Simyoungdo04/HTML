정규표현식

정규표현식(Reguler Expression : REGEX / REGEXP) 이란,
특정 패턴을 가진 문자열을 찾거나 찾아서 변경할 떄 사용하는 형식 언어
String 클래스 => mathes()
정규표현식을 사용하면 조건검사를 하거나 값을 변경 할 때
조건문 / 반복문을 사용하지 않고 간단히 처리할 수 있음
자바스크립트 뿐만 아니라 자바/오라클에서도 사용할 수 있음

* 메타문자

메타문자를 사용해서 "검색하고자 하는 패턴"을 생성할 수 있음

* 플래그문자

/정규표현식/ 뒤에써줌

종류
i : 대소문자 전부검사
g : 문자열 내의 모든 패턴 검사(전역으로 보겠다)
m : 여러줄을 전부 검사

정규표현식을 이용해서 주민등록번호 검사
