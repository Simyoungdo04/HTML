브라우저에서 제공하는 인터페이스

브라우저 상에서 자바스크립트라는 언어로 브라우저라는 프로그램을 제어할 수 있도록 BOM과 DOM이라는 인터페이스가 제공됨

- BOM(Browser Object Model) : 브라우저 관련
- DOM(Document Object Model) : HTML문서 관련
